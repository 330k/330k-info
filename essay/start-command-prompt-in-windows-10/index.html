Windows 10でコマンドプロンプトを素早く起動する方法

July 17, 2018 (Last Modified: May 18, 2020)
雑記帳

元ネタ: Windows 10でコマンドプロンプトを素早く起動する方法

私が業務で使うときは以下の手順で起動している。 設定が不必要で、おそらく最速。

管理者権限でコマンドプロンプトを起動する

  1. [Windows] + [X] → [A]で管理者権限でPowerShellを起動
  2. UACのダイアログが出たら許可する
  3. PowerShell上で「cmd」と入力してEnter。

UACがONのときでもキーストローク数は9ストローク。

コマンドプロンプトを起動しなくても、

  • コマンドプロンプトとほぼ同じ操作ができるようにPowerShellのAliasが設定されている
  • 将来性を考えたらPowerShellに慣れていたほうが得

と思うので、PowerShellを起動するだけで良いのであれば、キーストローク数は5ですむ。
